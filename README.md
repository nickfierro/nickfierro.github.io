# simplescreen
Start geth node with:
```
geth --rpc --rpccorsdomain "http://localhost:8080"

```

Install node modules:
```
npm install
```

Install webpack globally to use CLI and node http-server:
```
sudo npm install -g webpack http-server
```

Run webpack
```
weback
```
Start http-server:

```
http-server
```
